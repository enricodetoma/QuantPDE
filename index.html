What is QuantPDE?

QuantPDE is a high-performance, open-source, C++ library for pricing derivatives. The library comes with a host of ready-to-use pricing routines.

Particular focus is placed on superlinearly convergent methods for solving optimal control problems (e.g. American options).

QuantPDE uses sparse linear solvers provided by Eigen, a template library for linear algebra.
